This page will tell you how colors work in CSS

Colors are easely made by CSS, you just need a color in mind and a stylesheet in your coding editor.

to make a background color you need to add a class or id to the page you want changed, then type this!

.only { background-color: beige; }
Same goes for color!
xmp { color: chartreuse; display: block; border-color: white; background-color: black; border-style: outset; border-radius: 30px; }
